Initially the values for a, b and c were set to 1, which produced an upward-opening parabola that did not match the data at all. By observing the scatter plot, it was clear the parabola must open downward (negative a), have a y-intercept near 0 (c close to 0), and its peak must occur at a positive x value (positive b). After adjusting, a = -0.8, b = 3.0, and c = -2.0 produced a parabola that closely approximates the data points.
