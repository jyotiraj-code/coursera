Damped Harmonic Oscillator: f(t) = A·e^(−γt)·cos(ωt)¶

In [ ]:
A = 1.0
γ = 0.3
ω = 2π

f(t) = A * exp(-γ * t) * cos(ω * t)

About this Function¶

This function models a damped harmonic oscillator a system that oscillates while losing energy over time due to friction or resistance (e.g., a pendulum in air, a mass on a spring in a fluid).

The function is:

$$f(t) = A e^{-\gamma t} \cos(\omega t)$$

Where:

  • $A = 1.0$ is the initial amplitude
  • $\gamma = 0.3$ is the damping coefficient (controls how fast oscillations decay)
  • $\omega = 2\pi$ rad/s is the angular frequency (one full oscillation per second)

The exponential envelope $\pm A e^{-\gamma t}$ bounds the oscillation and decays to zero as $t \to \infty$. This is the underdamped regime, where the system oscillates before eventually coming to rest.

using Plots
gr()

t_vals = collect(0:0.01:15)
y_vals = f.(t_vals)
envelope_upper =  A .* exp.(-γ .* t_vals)
envelope_lower = -A .* exp.(-γ .* t_vals)

plot(t_vals, y_vals,
    label = "f(t) = e^{-0.3t}·cos(2πt)",
    linewidth = 2,
    color = :blue,
    title = "Damped Harmonic Oscillator",
    xlabel = "Time (s)",
    ylabel = "Amplitude",
    legend = :topright
)
plot!(t_vals, envelope_upper, linestyle = :dash, color = :red, label = "Envelope ±e^{-0.3t}", linewidth = 1)
plot!(t_vals, envelope_lower, linestyle = :dash, color = :red, label = "", linewidth = 1)
